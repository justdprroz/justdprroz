## Current Project

Currently I'm working on [Window manager for X server](https://github.com/justdprroz/rtde) written in Rust! Intention of this project is to replicate [DWM](https://dwm.suckless.org) in Rust with learning how window manager works and how to implement some system-level stuff in Rust. My window manager currently in pre-release state so you can check it out.

Presenting this project I won:
* [HSE project conference](https://olymp.hse.ru/projects/cs-avangard)
* 2nd degree award on [Baltic Science and Engineering contest](https://baltkonkurs.ru/) in Saint Petersburg in 2024
